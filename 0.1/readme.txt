== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the folder 'burstn' to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to appearance -> widgets and drag the "latest burstn photos" widget into one of your widget ready areas.
4. Specify the Widget name, your burstn username and the number of photos to show.
5. Get burstn!
